## Missions

### Mission 1: Foundations

**Core Subjects**: Intro programming (Python), discrete math (proofs, sets), basic data structures (arrays, lists).[^2][^1]

**Practical Tasks**:

- Build a command-line todo app with file persistence.
- Solve 50 easy LeetCode problems on arrays/strings.

**Daily Developer Habits**:

- 30 mins LeetCode/HackerRank for algorithms.
- Read 1 research paper abstract (arXiv CS.AI).
- Commit code to personal GitHub repo with PR simulation.


### Mission 2: Core Systems \& SE

**Core Subjects**: OOP/design patterns, algorithms (sorting, graphs, DP), OS/networking, databases (SQL/NoSQL).[^1][^2]

**Practical Tasks**:

- Implement Nand2Tetris CPU/simulator from gates.
- Build a RESTful web API (Node.js/Express) with auth and DB.

**Daily Developer Habits**:

- 20 mins code review on GitHub open-source repos.
- Pair program 15 mins via Discord/Pramp.
- Journal/refactor 10 mins of yesterday's code.


### Mission 3: Advanced SE

**Core Subjects**: Software architecture, testing/debugging, distributed systems, DevOps (CI/CD, Docker, Kubernetes).[^4][^1]

**Practical Tasks**:

- Deploy a microservices app to AWS/GCP with monitoring.
- Conduct full test suite/capstone project (e.g., OSSU software dev).

**Daily Developer Habits**:

- 30 mins LeetCode medium/hard problems.
- Read 1 research paper (Google Scholar AI/SE).
- Contribute 1 PR/review to open-source weekly.


### Mission 4: AI/ML Fundamentals

**Core Subjects**: Linear algebra/probability, ML (supervised/unsupervised, neural nets), PyTorch/TensorFlow basics.[^5][^1]

**Practical Tasks**:

- Train image classifier on MNIST/CIFAR-10 dataset.
- Build sentiment analyzer with scikit-learn.

**Daily Developer Habits**:

- 20 mins Kaggle competitions/datasets.
- Shadow 1 AI tutorial (fast.ai/Hugging Face).
- Log experiments in MLflow/Weights \& Biases.


### Mission 5: Modern AI Specialization

**Core Subjects**: Transformers/LLMs, fine-tuning (PEFT/LoRA), RAG (LangChain/LlamaIndex), cloud APIs (OpenAI, Vertex AI).[^6][^5]

**Practical Tasks**:

- Fine-tune Llama-2 on custom dataset for Q\&A bot.
- Implement RAG pipeline with vector DB (Pinecone) and deploy to cloud.

**Daily Developer Habits**:

- 30 mins Hugging Face models experimentation.
- Read 1 arXiv paper on GenAI/RAG.
- Prototype 1 micro-feature (e.g., API integration).

# English roadmap
This roadmap follows the CEFR framework from A1 beginner to C1 advanced (IELTS 7.5+ equivalent), drawing on official descriptors from Cambridge English, British Council, and aligned syllabi from Oxford. [cambridgeenglish](https://www.cambridgeenglish.org/exams-and-tests/cefr/)

## Missions
Sequential phases build progressively, with each requiring about 200 guided learning hours per CEFR level. [support.cambridgeenglish](https://support.cambridgeenglish.org/hc/en-gb/articles/202838506-Guided-learning-hours)

### Phase 1: A1-A2 Basics
- Learn 300-500 high-frequency words (greetings, family, numbers, daily activities). [yourenglishpal](https://yourenglishpal.com/grammar/english-grammar-cefr-levels/)
- Master present simple, to be, basic questions, and there is/there are. [scribd](https://www.scribd.com/document/996230435/English-Syllabus-A1-to-C2)
- Introduce yourself, ask/answer simple personal questions, and describe routines. [learningenglishwithoxford.oup](https://learningenglishwithoxford.oup.com/2023/05/05/what-cefr-level-are-you/)
- Understand slow, clear instructions and basic phrases in listening/reading. [slideshare](https://www.slideshare.net/slideshow/self-assessmentgriden/15289669)

### Phase 2: B1 Foundations
- Build to 1,500 words including topics like education, health, travel. [scribd](https://www.scribd.com/document/996230435/English-Syllabus-A1-to-C2)
- Practice present perfect, past simple, modals (can, should, must), and first/second conditionals. [examenglish](https://www.examenglish.com/CEFR/cefr_grammar.htm)
- Handle everyday conversations, write simple emails, and express opinions. [virtualenglishcourse](https://virtualenglishcourse.com/post/cambridge-english-a2-to-c1-guide)
- Comprehend main ideas in clear texts and short narratives. [learnenglish.britishcouncil](https://learnenglish.britishcouncil.org/level/understand-your-english-level)

### Phase 3: B2 Intermediate
- Expand vocabulary to 3,000+ words (media, environment, business). [examenglish](https://www.examenglish.com/CEFR/cefr_grammar.htm)
- Master passives, reported speech, relative clauses, and mixed conditionals. [scribd](https://www.scribd.com/document/996230435/English-Syllabus-A1-to-C2)
- Debate familiar topics, write essays/reports, and understand detailed audio. [virtualenglishcourse](https://virtualenglishcourse.com/post/cambridge-english-a2-to-c1-guide)
- Use linking words for coherent speech and texts. [cambridgeenglish](https://www.cambridgeenglish.org/images/168062-cambridge-english-first-for-schools-fce-for-schools-writing-assessment-scales.pdf)

### Phase 4: C1 Breakthrough
- Acquire advanced/academic vocabulary, idioms, phrasal verbs (4,000+ words total). [examenglish](https://www.examenglish.com/CEFR/cefr_grammar.htm)
- Command inversion, emphasis, advanced passives, and nuanced modals. [examenglish](https://www.examenglish.com/CEFR/cefr_grammar.htm)
- Produce fluent, structured arguments; understand implicit meanings in complex texts. [cambridgeenglish](https://www.cambridgeenglish.org/images/168062-cambridge-english-first-for-schools-fce-for-schools-writing-assessment-scales.pdf)
- Simulate IELTS 7.5+ tasks: detailed essays, spontaneous discussions. [ieltsidpindia](https://ieltsidpindia.com/blog/what-is-a1-a2-b1-b2-c1-c2-in-ielts)

## Routines
These daily/weekly habits are backed by research on acquisition (e.g., comprehensible input) and retention (e.g., spacing effect), recommended by Cambridge and British Council resources. [curiositybug](https://curiositybug.com/learning/spaced-repetition-language-learning/)

- Spaced repetition flashcards (Anki/Duolingo): Review 20-50 items daily for 15-20 mins to boost retention 200-400%. [facebook](https://www.facebook.com/groups/616774812293099/posts/807178029919442/)
- Comprehensible input (podcasts/graded readers at level): 30 mins/day listening/reading for natural acquisition. [cambridge](https://www.cambridge.org/core/journals/journal-of-classics-teaching/article/comprehensible-input-and-krashens-theory/2308987050E8D31E3986B530D4B02F6F)
- Shadowing (repeat audio scripts): 10-15 mins/day to improve fluency/pronunciation. [ielts.idp](https://ielts.idp.com/vietnam/about/news-and-articles/article-ielts-roadmap-to-score-7-5/en-gb)
- Active recall/output (speak/write journal): 15 mins/day describing photos/events without notes. [facebook](https://www.facebook.com/groups/616774812293099/posts/807178029919442/)
- Immersion (English media/news): 20-30 mins/day, weekly conversation practice for contextual use. [learnenglish.britishcouncil](https://learnenglish.britishcouncil.org/level/understand-your-english-level)

- Estimated time: 250–350 guided hours (approx. 3–6 months at 15–20 hrs/week) to cross A2→B1 per CEFR and Cambridge recommendations. [cambridgeenglish](https://www.cambridgeenglish.org/Images/126011-using-cefr-principles-of-good-practice.pdf)

- Authoritative resources:
  - Cambridge English B2 descriptors and practice materials (FCE-level resources). [cambridgeenglish](https://www.cambridgeenglish.org/images/168062-cambridge-english-first-for-schools-fce-for-schools-writing-assessment-scales.pdf)
